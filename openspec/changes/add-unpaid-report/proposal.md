## Why

Пользователям необходимо отслеживать выполненные задачи, по которым ещё не выставлено время к оплате. Это критично для фрилансеров и консультантов, которым важно контролировать неоплаченную работу и своевременно выставлять счета клиентам.

## What Changes

- Добавлен новый отчёт "Неоплаченные задачи" в разделе аналитики
- Отчёт показывает только задачи со статусом "Выполнено" (completed), у которых есть сегменты времени с ненулевым фактическим временем
- Колонки отчёта: Задача, Дата окончания, Продолжительность (факт, минуты), Продолжительность к оплате (минуты)
- Фильтры: по Проекту (все/выбранный), по Клиенту (все/выбранный), по Списку задач (все/выбранный)
- Кнопка экспорта отчёта в формат ODS (OpenDocument Spreadsheet)
- Новый endpoint API для получения данных отчёта
- Новый endpoint API для экспорта в ODS

## Impact

- **Backend**: новый endpoint `GET /api/analytics/unpaid-tasks` с параметрами фильтрации; новый endpoint `GET /api/analytics/unpaid-tasks/export` для ODS-экспорта
- **Backend**: зависимость `odfpy` для генерации ODS-файлов
- **Frontend**: новый компонент `unpaid-tasks-report` в разделе аналитики
- **Frontend**: добавление кнопки перехода к отчёту в навигации аналитики
- **Database**: запрос использует существующие таблицы `tasks`, `time_segments`, `projects`, `clients`, `task_lists` — изменений схемы не требуется
