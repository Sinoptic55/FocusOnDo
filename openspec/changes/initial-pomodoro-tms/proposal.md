## Why

Создаётся система управления задачами с интегрированным таймером Помодоро для решения двух ключевых проблем: личной продуктивности (борьба с прокрастинацией, понимание куда уходит время) и деловой отчётности (знать сколько времени потрачено на проект или клиента, сколько выставить в счёте). Текущая кодовая база пуста — это начальная реализация всей системы.

## What Changes

- Создание полнофункциональной веб-системы управления задачами с учётом времени
- Реализация техники Помодоро с автоматическим трекингом времени на задачи
- Три вида визуализации задач: списки, доска по дате, доска по статусу
- Иерархическая структура задач с подзадачами (шагами)
- Умные инструменты для планирования и избегания застревания
- Система аналитики продуктивности и учёта времени по проектам/клиентам
- Аутентификация и хранение данных на сервере
- Технологический стек: TypeScript/JavaScript/HTML фронтенд, Python бэкенд, PostgreSQL база данных

## Capabilities

### New Capabilities

- `task-management`: Управление задачами — CRUD операции, иерархическая структура (задачи → шаги), все поля задачи (название, описание, даты, статус, проект, клиент, оценка, первое действие, внешняя ссылка)
- `task-views`: Три вида отображения задач — списки с цветовой группировкой, доска по дате (6 колонок), доска по статусу (настраиваемые колонки)
- `pomodoro-timer`: Таймер Помодоро — настраиваемая последовательность интервалов, управление (старт/пауза/следующий), привязка к активной задаче, вопросы после интервала (прогресс, уровень энергии)
- `time-tracking`: Автоматический учёт времени — фиксация времени начала и длительности работы, разделение на фактическое и выставленное время, real-time отображение суммарного времени
- `reference-data`: Справочники — управление списками задач (с цветами), статусами (с видимостью на доске), проектами (с цветами), клиентами, интервалами таймера
- `recurring-tasks`: Повторяющиеся задачи — настройка периодичности (день/неделя/месяц), автоматическое создание следующей задачи при завершении
- `smart-tools`: Умные инструменты — быстрый захват задач (горячая клавиша), автовыбор приоритетной задачи ("не знаю с чего начать"), утренний ритуал планирования, недельный обзор, детекция паттерна застревания
- `analytics`: Аналитика продуктивности — время по проектам/клиентам (фактическое vs выставленное), пики продуктивности по времени дня, точность оценки задач, скорость работы, паттерны застревания
- `user-auth`: Аутентификация — система логина/пароля, управление сессиями, защита данных пользователя
- `settings`: Настройки приложения — темы (светлая/тёмная/авто), звуковые уведомления (раздельно для работы и отдыха), настройка горячей клавиши, переключатели для умных инструментов, порог детекции застревания

### Modified Capabilities

Нет существующих capabilities для модификации — это новый проект.

## Impact

**Новая кодовая база**:
- Frontend: TypeScript/JavaScript/HTML приложение с тремя режимами отображения
- Backend: Python веб-сервер с REST API
- Database: PostgreSQL схема для задач, времени, справочников, пользователей, аналитики
- Authentication: Система логина с защитой маршрутов

**Зависимости**:
- Frontend: TypeScript компилятор, возможно React или Vue для UI
- Backend: Python web framework (Flask/FastAPI), PostgreSQL драйвер
- Database: PostgreSQL 12+

**Системные требования**:
- Windows Server для развёртывания
- Браузерный доступ для пользователей
- Будущая поддержка: Windows desktop приложение, мобильные клиенты
