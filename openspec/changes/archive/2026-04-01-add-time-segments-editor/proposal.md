## Why

В форме задачи отсутствует быстрый доступ к просмотру и редактированию всех сегментов времени. Пользователям необходимо видеть историю затраченного времени прямо из формы задачи, чтобы управлять фактическим и выставленным временем, корректировать данные и удалять ошибочные записи.

## What Changes

- Добавление кнопки-иконки рядом с заголовком секции "Основное" в форме задачи
- При нажатии открывается диалог с таблицей всех сегментов времени текущей задачи
- Таблица отображает все поля `time_segments`: id, start_time, duration_seconds, actual_time_seconds, billed_time_seconds, energy_level, task_progressed, stuck
- Возможность редактирования, добавления и удаления записей прямо в диалоге
- Дизайн диалога соответствует стилю приложения (использует те же CSS-классы и компоненты)

## Capabilities

### New Capabilities
- `time-segments-editor-dialog`: Диалог редактирования сегментов времени, открываемый из формы задачи. Включает таблицу со всеми полями time_segments и CRUD-операциями.

### Modified Capabilities
- `task-management`: Форма задачи получает новую кнопку для открытия редактора сегментов времени

## Impact

- `src/frontend/src/components/task-form.ts` — добавление кнопки рядом с заголовком "Основное" и обработчика открытия диалога
- `src/frontend/src/components/time-segments-list.ts` — возможно расширение существующего компонента для отображения всех полей (task_progressed, stuck, duration_seconds)
- `src/frontend/src/components/dialog.ts` — повторное использование для создания нового диалога
- `src/frontend/styles.css` — возможные стили для новой кнопки и таблицы
