# Changelog

Все заметные изменения в этом проекте будут задокументированы в этом файле.

## [1.0.0] - 2026-03-23

### Добавлено
- Начальная реализация Pomodoro TMS (Task Management System).
- Backend на FastAPI с поддержкой JWT аутентификации.
- База данных PostgreSQL с миграциями Alembic.
- Модели для задач, проектов, клиентов, списков, статусов, интервалов помодоро и повторяющихся задач.
- Полнофункциональный API для управления задачами и временем.
- Аналитические отчеты по проектам, клиентам, продуктивности и застреваниям.
- Frontend на чистом TypeScript (без фреймворков).
- Три вида отображения задач: Списки, Доска по датам, Доска по статусам.
- Интегрированный таймер Помодоро с сохранением состояния.
- Умные инструменты: "Утренний ритуал", "Недельный обзор", "Автовыбор следующей задачи".

### Исправлено
- Исправлена ошибка `NameError: name 'String' is not defined` в модели `RecurringTask`.
- Исправлена ошибка `NameError: name 'Depends' is not defined` в API аутентификации.
- Исправлена несовместимость `passlib` с новыми версиями `bcrypt` (переход на прямое использование `bcrypt`).
- Исправлена ошибка импорта `Optional` из `pydantic` в схемах задач и временных отрезков.
- Исправлена ошибка группировки задач в `DateBoardView` (теперь колонка "Позже" корректно заполняется).
- Исправлена ошибка инициализации коллбэков в компоненте `Timer`.
- Исправлено отсутствие обязательных полей `updated_at` в ответах API задач при создании.

### Тестирование
- Добавлен набор юнит-тестов для аутентификации и JWT.
- Добавлены интеграционные тесты для API задач и аналитики.
- Добавлены тесты логики повторяющихся задач.
- Добавлены скрипты для ручного тестирования в условиях отсутствия `pytest`.
